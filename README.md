# PromptVault

PromptVault is a curated collection of professional prompts designed for advanced Large Language Models (LLMs), including ChatGPT, Claude and Gemini.

The repository is organized by domain, allowing each prompt to evolve independently while maintaining proper version control, documentation and change history.

## Repository Structure

Each domain contains one or more prompt collections focused on a specific subject.

```text
PromptVault/
├── finance/
│   └── financial-analysis/
├── README.md
├── CHANGELOG.md
├── LICENSE
└── .gitignore
```

## Objectives

The project aims to:

- Develop reusable, production-quality prompts.
- Apply software engineering practices to prompt engineering.
- Preserve the complete evolution of each prompt through Git.
- Maintain bilingual versions whenever appropriate.
- Document changes using Semantic Versioning and Conventional Commits.

## Development Principles

Each prompt is developed following an iterative process:

1. Initial implementation.
2. Structural improvements.
3. Professional refinement.
4. Validation and testing.
5. Stable release.

Every major iteration is preserved in the repository to document the evolution of the prompt.

## Repository Organization

Each prompt collection is self-contained and typically includes:

- Prompt files.
- README.
- CHANGELOG.
- Version history.
- Supporting documentation when required.

## Version Control

PromptVault follows:

- Semantic Versioning
- Conventional Commits
- Incremental development

This approach allows every significant improvement to be tracked and documented independently.
